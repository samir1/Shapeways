    Describe the pros and cons of version control.

    Version control offers a system to keep track of file changes. Some of the pros of version control are the ability to keep track of file changes, keep track of who changes a file, reverting back to a previous version of a file, and being able to see the history of a project. There are also a couple problems to version control: it requires a system to keep track of changes, keeping track of a lot of small changes can add additional time to the process, and learning a version control system can have a steep learning curve.
